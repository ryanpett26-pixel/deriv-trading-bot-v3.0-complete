# Deriv Trading Bot v3.0 Complete

## Description
This project features a trading bot designed to work seamlessly with the Deriv platform. Equipped with a dashboard for analytics and a plug-and-play setup.

## Features
- Bot trading strategies
- Real-time monitoring dashboard
- Modular scripts for customization

## Setup and Installation
1. Clone the repo:
   ```
   git clone <repo-url>
   ```
2. Update configuration files in `config/` and set environment variables in `.env`.
3. Build and run the Docker container:
   ```
   docker-compose up --build
   ```

## Usage
- Use `run_bot.sh` to execute the bot.
- Access the dashboard at `http://localhost:5000` (default).
